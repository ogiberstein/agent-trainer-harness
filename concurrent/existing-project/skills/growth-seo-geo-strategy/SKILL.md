# Growth SEO GEO Strategy

## Purpose
Build an actionable, measurable organic discovery strategy across search engines and generative answer surfaces.

## Use When
- Launching a new product/site section.
- Improving organic acquisition for existing features.
- Planning content structure for discoverability.

## Inputs Required
- `BRIEF.md`
- `specs/requirements.md`
- `docs/README.md` and key public docs/pages
- Known audience segments and geographic targets

## Steps
1. Define target intents by user segment and funnel stage.
2. Build keyword/topic clusters and map each to a page type.
3. Specify content structure and internal linking strategy.
4. Define technical prerequisites (metadata/schema/performance/indexability).
5. Define GEO visibility tactics (entity clarity, concise answer blocks, trust signals).
6. Set KPIs and reporting cadence.

## Output Contract
- Growth section for organic strategy in `specs/growth-plan.md`
- Prioritized backlog with owner, impact estimate, and KPI per item

## Safety and Security
- Do not fabricate performance claims.
- Keep recommendations compliant with brand/legal constraints.
