# Progress Log

Running log of what was built, what broke, and how it was fixed. Write entries as you go -- this is a working document, not a formal report. Future agents and humans read this to understand the *journey*, not just the destination.

## Format

```
### YYYY-MM-DD — [short title]

**What:** [what was done]
**Why:** [what prompted it]
**Gotchas:** [what broke, root cause, how it was fixed]
**Notes:** [anything useful for the next person]
```
